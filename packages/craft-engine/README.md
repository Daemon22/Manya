# Craft Engine

Craft Engine is a powerful 7-fold compression and encryption engine designed for optimal data handling within the Manya ecosystem.

## Features

- **7-fold Compression**: Achieves high compression ratios to minimize data footprint.
- **Robust Encryption**: Ensures data security with advanced encryption standards.
- **Seamless Integration**: Designed to work efficiently with other Manya components.

## Installation

This package is part of the Manya monorepo. To install all dependencies, navigate to the root of the Manya repository and run:

```sh
npm install
```

## Usage

For usage details and API reference, please refer to the main Manya documentation and the source code within this package.

## Testing

To run tests for Craft Engine, navigate to the root of the Manya repository and execute:

```sh
npm run packages:test
```
