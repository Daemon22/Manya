# @manya/helixflow-sdk

Small SDK for HelixFlow workflow automation.

This package helps developers:

- Create workflow definitions.
- Validate basic DAG shape before sending workflows to HelixFlow.
- Run workflows through the HelixFlow API.
- Use uSINGA connection references for API request nodes.

It intentionally does not store API keys, inspect provider balances, or choose providers. Those jobs belong to `uSINGA - API NEXUS`.
