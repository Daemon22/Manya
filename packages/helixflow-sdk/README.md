# HelixFlow SDK

The HelixFlow SDK provides client-side utilities and workflow helpers for interacting with the HelixFlow visual workflow orchestration tool within the Manya ecosystem.

## Features

- **Client for HelixFlow**: Facilitates communication with the HelixFlow API.
- **Workflow Helpers**: Simplifies common workflow operations and integrations.
- **Monorepo Integration**: Designed for seamless use within the Manya monorepo.

## Installation

This package is part of the Manya monorepo. To install all dependencies, navigate to the root of the Manya repository and run:

```sh
npm install
```

## Usage

Refer to the main Manya documentation and the HelixFlow tool documentation for detailed usage instructions and API examples.

## Testing

To run tests for the HelixFlow SDK, navigate to the root of the Manya repository and execute:

```sh
npm run packages:test
```
